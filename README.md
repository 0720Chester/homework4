基于 Taichi 框架实现的交互式 Phong 光照渲染项目，完成计算机图形学实验要求，包含球体、圆锥隐式建模、光线投射求交、深度测试及实时 UI 参数调节功能。
 实验目标
- 理论理解：掌握局部光照基本原理，区分环境光（Ambient）、漫反射（Diffuse）和镜面高光（Specular）。
- 数学基础：熟练运用三维空间向量运算（法向量、光线方向、视线方向、反射向量）。
- 工程实践：使用 Taichi 实现交互式渲染，通过 UI 控件实时调节材质参数，观察参数对渲染效果的影响。
   核心功能
- 隐式几何体建模：实现红色球体、紫色圆锥两个几何体，无需外部模型文件。
- 光线投射与深度测试：模拟 Z-buffer 逻辑，处理物体遮挡关系，选择最近交点进行着色。
- Phong 光照渲染：严格按照 Phong 模型公式，计算环境光、漫反射、镜面高光三个分量并叠加。
- 实时 UI 交互：通过滑动条调节环境光系数、漫反射系数、镜面高光系数及高光指数，实时预览渲染效果。
   环境配置
依赖库安装

 场景与参数说明
固定场景参数
- 摄像机位置：(0, 0, 5)
- 点光源位置：(2, 3, 4)，光源颜色：纯白光 (1.0, 1.0, 1.0)
- 背景颜色：深青色 (0.0, 0.2, 0.3)
几何体参数
- 红色球体：圆心 (-1.2, -0.2, 0)，半径 1.2，颜色 (0.8, 0.1, 0.1)
- 紫色圆锥：顶点 (1.2, 1.2, 0)，底面高度 y=-1.4，底面半径 1.2，颜色 (0.6, 0.2, 0.8)
可调节参数（UI 滑动条）
- Ka（环境光系数）：0.0 ~ 1.0，默认值 0.2
- Kd（漫反射系数）：0.0 ~ 1.0，默认值 0.7
- Ks（镜面高光系数）：0.0 ~ 1.0，默认值 0.5
- Shininess（高光指数）：1.0 ~ 128.0，默认值 32.0
   核心代码说明
- 光线求交：实现 ray_sphere_intersect（光线-球体求交）、ray_cone_intersect（光线-圆锥求交）函数。
- 法向量计算：实现 get_sphere_normal、get_cone_normal 函数，获取几何体表面法向量。
- Phong 着色：phong_shading 函数，按公式计算三个光照分量并叠加，得到最终像素颜色。
- 渲染内核：render 函数，实现光线投射、深度测试、着色逻辑，渲染整个画面。
   注意事项
- 运行时若出现卡顿，可降低画布分辨率（修改代码中 WIDTH、HEIGHT 参数）。
- 若 PyCharm 上传 GitHub 失败，可直接通过 GitHub 网页上传 phong_renderer.py 和 README.md 文件。
- 确保 Taichi 版本为最新，避免因版本兼容问题导致渲染异常。
